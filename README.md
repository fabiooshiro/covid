# COVID
Projeto para validar/invalidar algumas hipóteses.  

Leitor de dados de óbitos no Brasil.  
Arquivo zip foi baixado do site:
http://svs.aids.gov.br/dantps/centrais-de-conteudos/dados-abertos/sim/

Para utilizar descompacte o zip, instale as dependências:

```
yarn install
```

E rode o projeto:
```
yarn dev
```
